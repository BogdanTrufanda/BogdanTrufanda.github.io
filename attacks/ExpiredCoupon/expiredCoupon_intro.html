Expired Coupon - Intro

About

Expired coupon is a four star challenge from OWASP Juice Shop. In order to solve this challenge you have to redeem an expired coupon from old campaigns while doing a checkout for an order.

Being an Improper Input Validation vulnerability, when software does not validate input properly, an attacker is able to craft the input in a form that is not expected by the rest of the application. This will lead to parts of the system receiving unintended input, which may result in altered control flow, arbitrary control of a resource, or arbitrary code execution.

As the title of this challenge says, we should be able to redeem an expired coupon while doing the checkout of our order.The problem is that we do not know how that coupon looks like and how to redeem it.
