Forged Review - Attack

The first step to take in order to complete this type of attack is to login with an existing user and post a review while monitoring with Burpsuite.

As we can see above, we have a PUT request with our review and our email.

In this case we can assume that we could change the fields within PUT request with another user's email and review.

In this case we will get the PUT request and send it to Repeater in BurpSuite in order to modify the values of the fields.

You can see that the PUT request is successfull meaning that we have posted a review in with someone's email as author without having access to that email.

If we are checking the reviews of that product, we can see our false review.
