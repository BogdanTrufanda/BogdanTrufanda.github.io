Upload a file that has no .pdf or .zip extension

Upload a file that has no .pdf or .zip extension In addition to the maximum file size, the Juice Shop also verifies that the uploaded file is actually a PDF. All other file types are rejected.

  • Craft a POST request to https://juice-shop.herokuapp.com/file-upload with a form parameter file that contains a non-PDF file with a size of less than 200 kB.
  • The response from the server will be a 204 with no content, but the challenge will be successfully solved.
