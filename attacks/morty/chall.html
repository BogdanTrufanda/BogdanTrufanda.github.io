Morty password

Reset Morty's password via the Forgot Password mechanism.

  • Go to reset password page and see that the security answer is Morty`s dog name!
  • Create a word list of all mutations (including typical "leet-speak"-variations!) of the strings snuffles and snowball using only lower case (a-z) upper case (A-Z) and digit characters (0-9)
  • Write a script that iterates over the word list and sends well-formed requests to http://juiceshop:port/rest/user/reset-password. A rate limiting mechanism will prevent you from sending more than 100 requests within 5 minutes, severely hampering your brute force attack.
  • Change your script so that it provides a different X-Forwarded-For-header in each request, as this takes precedence over the client IP in determining the origin of a request.
  • Rerun your script you will notice at some point that the answer to the security question is 5N0wb41L and the challenge is marked as solved.
